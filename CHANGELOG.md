# Changelog

## [1.0.2]

- **密码生成 (PasswordGenerator)**：
  - 修复了修改选项（如长度滑块）时会触发两次重复密码生成的问题。
- **SQL IN 构造 (SqlInBuilder)**：
  - 优化了界面交互性能，采用结果缓存策略消除了输入或配置改变时的重复计算，降低资源消耗。
- **系统核心 (Core)**：
  - 重构并强化了后台文件监控（Watcher）启动机制的单元测试用例，提升了代码库长期维护的稳定性。

## [1.0.1]

- **文本对比 (FileDiff)**：
  - 新增输入文件大小限制 (10MB)，防止浏览器原生拖拽等方式导致的内存溢出 (OOM)。
  - 新增超大文件 (>10000行) 的降级算法，由底层全量比对切换为逐行极速比对，并在界面顶部增加明显的大文件降级警告提示。
  - 修复多处硬编码的英文报错，并实现报错信息的完整国际化（支持中英文）。
  - 优化高亮差异字词的内部渲染性能。
- **JSON 格式化 (JsonFormatter)**：
  - 在「原始输入」面板左侧新增代码行号显示列，方便定位报错中的语法错误。
  - 在「格式化输出」面板中新增了**嵌套节点折叠与展开功能**，并精确对齐视觉缩进。
  - 修复了 `escapeHtml` 无法正确全局转义长字符串带 `<`、`>` 符号的问题。
- **其他维护**：
  - 修复 `App.tsx` 中关于自动更新信息 `updater.updateInfo?.version` 缺少依赖的 React 警告。
